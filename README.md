# DAMXd89 ULTIMATE BOT v5.3 🚀

A powerful Telegram bot for CC Checking, BIN Lookup, CC Generation, Auto Hitter (Stripe), and more.

## Features
- 💳 **CC Checker (VBV/3DS)**
- 🔍 **BIN Lookup**
- 🔥 **Auto Hitter (Stripe Checkout)**
- 🧹 **CC Cleaner**
- 🛡️ **Captcha Solver**
- 💰 **Payment Integration** (Cashfree & Crypto)
- 🎁 **Redeem Code System**
- 📢 **Share & Win System**

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your `.env` file with the following keys:
   - `BOT_TOKEN`
   - `ENDER_KEY`
   - `CASHFREE_CLIENT_ID`
   - `CASHFREE_CLIENT_SECRET`
   - `NOWPAYMENTS_API_KEY`
   - `ADMIN_USER_ID`
4. Start the bot: `npm start`

## Deployment to Render
This project includes a `render.yaml` file for easy deployment as a Blueprint on Render.
